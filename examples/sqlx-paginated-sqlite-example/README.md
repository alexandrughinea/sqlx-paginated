# sqlx-paginated-sqlite-example

Test project demonstrating [sqlx-paginated](https://github.com/alexandrughinea/sqlx-paginated) with SQLite and Actix-web.

## Quick start

```bash
cargo run
```

Server starts at `http://127.0.0.1:8080`

Migrations run automatically on startup.

## API documentation

Use the **Bruno collection** in `docs/bruno/` for interactive API testing and documentation.

To use Bruno:
1. Install [Bruno](https://www.usebruno.com/)
2. Open the collection: `docs/bruno/`
3. Start the server: `cargo run`
4. Run requests from Bruno
